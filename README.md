# jd-mobile移动端页面骨架及项目目录结构
```
├─ /jd-mobile/ ··················· 项目所在目录
└─┬─ /css/ ······················· 我们自己的CSS文件
  ├─ /imgage/ ······················· 使用到的图片文件
  ├─ /js/ ························ 自己写的JS脚步
  ├─ /lib/ ······················· 从第三方下载回来的库【只用不改】
  └─ /index.html ················· 入口文件
```
